# 🚀 LeetCode DSA Solutions

A collection of solved Data Structures and Algorithms (DSA) problems from LeetCode. This repository serves as a record of my coding journey, focusing on efficient problem-solving and algorithmic thinking.

![LeetCode Stats](https://shields.io)
![Language](https://shields.io)

## 📁 Repository Structure

The solutions are organized by problem difficulty and topic for easy navigation:

- `/Easy` - Fundamental problems to build core concepts.
- `/Medium` - Intermediate problems focusing on optimization and complex data structures.
- `/Hard` - Advanced problems involving intricate algorithms and edge cases.
## 🛠️ Automated Syncing

This repository is automatically updated using the **[LeetSync](https://google.com)** browser extension. Every time a solution is "Accepted" on LeetCode, it is instantly pushed to this repo with the following:
- Problem name and number
- Cleanly formatted source code
- Problem description and constraints
- Runtime and memory performance stats

## 💡 Key Topics Covered
- **Arrays & Strings** (Two Pointers, Sliding Window)
- **Linked Lists** (Fast & Slow Pointers)
- **Trees & Graphs** (DFS, BFS, Dijkstra)
- **Dynamic Programming** (Memoization, Tabulation)
- **Sorting & Searching** (Binary Search, Merge Sort)

## 🚀 How to Use
1. Clone the repository: 
   `git clone https://github.com`
2. Navigate to a specific topic or difficulty folder to review the logic and implementation.
